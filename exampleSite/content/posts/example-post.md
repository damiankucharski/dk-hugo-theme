This is an example blog post demonstrating the DK Hugo Theme.

## Features Showcase

### Syntax Highlighting

The theme includes syntax highlighting for code blocks:

```python
def hello_world():
    print("Hello, World!")
    return True

if __name__ == "__main__":
    hello_world()
```

### LaTeX Support

Mathematical equations are supported using LaTeX:

Inline math: $E = mc^2$

Block math:

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

### Lists

Unordered list:
- Item one
- Item two
- Item three

Ordered list:
1. First item
2. Second item
3. Third item

### Blockquotes

> This is a blockquote. It can be used for highlighting important information or quotes from other sources.

### Links

You can create [links to other pages](/) or external websites.
