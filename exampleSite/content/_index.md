# Welcome to My Site

This is an example site demonstrating the DK Hugo Theme. This theme is designed for personal websites, blogs, and academic portfolios.

## Features

- Clean, responsive design
- Dark mode toggle
- Built-in search
- Syntax highlighting
- LaTeX support
- Tag taxonomy

## About Me

I'm a researcher and developer passionate about creating clean, functional websites. This theme showcases my work including projects, research papers, and blog posts.

Feel free to explore the site using the navigation menu above.
